# BedWars

## Controls
- **WASD:** Movement
- **Space:** Jump
- **Left Click:** Attack
- **Right Click:** Place Wool
- **E:** Open Shop *(must be less than 6 blocks from your spawn)*
- **Shift:** Sprint
- **Ctrl:** Slow Walk
- **Esc:** Pause Menu

### Ghost Movement
- **Space** Go Up
- **Shift** Go Down

## Information
> [!IMPORTANT]
> Swords register hits per collider enter.
